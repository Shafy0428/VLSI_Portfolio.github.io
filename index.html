About Me

Saifur Rahman

Software Engineer

  • LocationBangladesh
  • FocusFull-stack web development
  • AvailabilityOpen to opportunities

I'm a software engineer who enjoys turning ideas into working products — starting from a clear problem, sketching a simple solution, and building it into something reliable. I'm drawn to software engineering because it sits at the intersection of logic and craft: every feature is a small puzzle, and every clean solution feels like getting the details right.

My focus areas are full-stack web development, building accessible and responsive user interfaces, designing REST APIs, and working with relational and NoSQL databases. I pay close attention to code readability and maintainability, because software is read far more often than it's written.

My career goal is to grow into a developer who can own a problem end to end — from understanding what users actually need, to shipping a dependable, well-tested solution, to supporting it after launch. I'm currently looking for roles where I can keep learning from experienced engineers while contributing real value from day one.
